Key Components:
===============

1. Configuration (config.py)
   - API keys and settings
   - Browser configuration
   - Agent parameters

2. Claude Client (claude_client.py)
   - Interfaces with Anthropic API
   - Analyzes screenshots using vision
   - Makes intelligent decisions
   - Returns structured actions

3. Web Controller (web_controller.py)
   - Playwright browser automation
   - Executes actions from Claude
   - Takes screenshots
   - Handles popups and navigation

4. Shopping Agent (shopping_agent.py)
   - Main orchestrator
   - Implements shopping flow
   - Coordinates Claude + Web Controller
   - Manages state transitions
